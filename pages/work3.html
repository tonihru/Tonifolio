Habit Tracker App

Native Android Application

Semester: WS 2025 Role: Developer & UI Design Tools: Kotlin, Android Studio, Figma

A native Android application that allows users to track habits, visualize progress, and receive reminders. The project focused on usability and clean mobile UI design.

Project Screenshots

Overview

The goal of this project was to design a simple mobile tool that helps users build consistent daily habits.

Features

  • Create and track habits
  • Daily reminders
  • Progress visualization

Challenges

Managing navigation and persistent data storage was the main technical challenge during development.
